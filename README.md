# Timed LED Flow Control

An Arduino project which allows you to control the timed sequence of 8 lights based on color

## Materials

- Breadboard
- 8 LEDS
- 4 220ohm resistors
- Arduino Uno
- 2 Potentiometers
- Around 10 - 15 jumper wires
